# Beginner Workout Guide Website

## Project Purpose

The purpose of this project is to create a simple two-page website that provides beginners with a basic workout guide. The website includes an introduction to the project, a weekly workout schedule, basic exercises, and beginner fitness tips.

This project demonstrates basic project management, documentation, HTML, CSS, and GitHub Pages skills.
